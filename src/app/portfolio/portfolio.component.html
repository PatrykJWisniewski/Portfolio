DUNGEON CRAWLER

An individual project where I taught myself Unity, C#, and Javascript to create an alpha version of a dungeon crawler. Concepts I learned included UI Design, level design, lighting, and linear interpolation to overcome many design hurdles.

SPOTIFY SAFE PLAYLISTS

An individual project that uses Spotify’s API, along with Javascript and Node.JS, to create playlists of “safe” music. Safe in this case meaning music that would be considered mainstream enough that most people would enjoy and you would be comfortable sharing it. Working on the project in my free time and planning on incorporating filters to fine-tune playlists.

BROCCOLI'S JOURNEY

A group project where I mentored three people on how to use Unity to create a 2D platformer. Using Node.JS and firebase, I integrated the game into our web app. I also created 2D environments and a user-friendly UI for the game.
